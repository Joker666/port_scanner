# Port Scanner

A simple port scanner written in Python.

### Prerequisites

- Python 3.12.x
- Node.js 20.x
- pnpm

### Usage

1. Run `build.sh` to build the UI.
2. Run `sudo run.sh` to start the application.
3. Open `http://localhost:8080` in your browser.
